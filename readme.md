# Getting started

## Requirements
- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)

```bash
# Install the dependencies
$ npm install

# Finally, to the api service
$ npm start

# For development mode
$ npm run start:dev

# Well done, the aplication is up and running!
```
